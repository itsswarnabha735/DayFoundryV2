
  # Day Foundry Mobile App

  This is a code bundle for Day Foundry Mobile App. The original project is available at https://www.figma.com/design/aSsZa2DrLKrZPIr7k4OxLR/Day-Foundry-Mobile-App.

  ## Configuration

1. Copy `.env.example` to `.env`:
   ```bash
   cp .env.example .env
   ```
2. Open `.env` and add your Google Gemini API key:
   ```
   GEMINI_API_KEY=your_actual_api_key
   ```

## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.
